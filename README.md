# PDFMaker :page_facing_up:

![PDFMaker Banner](https://github.com/ACM-VIT/.github/blob/master/profile/forktober%202023%20new.png)

Welcome to PDFMaker, a handy tool that makes creating PDFs for your assignments a breeze. Whether you need to merge multiple documents, add a splash of color, or format text/code, PDFMaker has you covered!

## Getting Started :rocket:

1. **Fork** and **Clone** this repository to your local machine.
2. Navigate to the project directory.
3. Install the necessary dependencies by running:

   ```bash
   pip install -r requirements.txt
   ```

4. Start your Streamlit app with the following command:

   ```bash
   streamlit run app.py
   ```

5. You're all set to create PDFs effortlessly!

OR

Use the hosted version [here](https://pdfmaker.streamlit.app/).

## About Streamlit:

Streamlit is a Python library that turns data scripts into shareable web apps in minutes. No frontend experience is required. You can find more information in the [Streamlit Documentation](https://docs.streamlit.io/).
